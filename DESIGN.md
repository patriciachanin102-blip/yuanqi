# DESIGN.md

本项目采用 Stripe 风格的教育科技移动端设计语言：白底、深海军蓝文字、紫色主行动、多层蓝调阴影、精密数据卡片。

## Tokens
- Primary: `#533afd`
- Primary hover: `#4434d4`
- Heading: `#061b31`
- Body: `#64748d`
- Border: `#e5edf5`
- Surface: `#ffffff`
- Dark section: `#1c1e54`
- Success: `#15be53`
- Warning: `#f59e0b`
- Danger: `#ea2261`

## UI Rules
- 移动端优先，最大宽度 `430px`，模拟 App 壳体验。
- 标题使用轻字重与紧凑字距，数据使用等宽/等宽数字。
- 卡片使用 `rgba(50,50,93,0.25)` 与 `rgba(0,0,0,0.1)` 的多层阴影。
- 表单输入清晰、触控目标不小于 `44px`。
- 报告页强调可信来源、分数线、风险档位、录取概率。
